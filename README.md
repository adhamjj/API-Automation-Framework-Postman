# 🚀 API Automation Portfolio Project

## Overview
This project is an **Enterprise-Grade API Automation Framework** built with **Postman**. It demonstrates full CRUD operations, dynamic data chaining, and advanced validation techniques.

## 🛠️ Tech Stack
- **Postman** for API Request management.
- **JavaScript** for Test Scripts and dynamic variables.
- **JSON Schema** for structure validation.

## 📋 Features
- **Dynamic Chaining:** Automatically passing IDs between requests (Create -> Update -> Delete).
- **Schema Validation:** Ensuring response data types are correct.
- **Negative Testing:** Handling failed scenarios and error codes.
